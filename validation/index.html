Four-Tier Validation Infrastructure

The framework defines 31 acceptance criteria across four verification tiers with thresholds defined a priori.

TierFocusCriteriaKey Thresholds
Tier 1Unit6100% pass on synthetic tests
Tier 2Component8Variance reduction ≥ 30%, d' ≥ 1.50
Tier 3System10AUC ≥ 0.95, EER ≤ 0.08, Rank-1 ≥ 0.90
Tier 4Operational7FAR drift < 0.1%/month, throughput ≥ 100/s
