# Contributing

Thank you for your interest in contributing to the AnubisX Framework.

## Research Contributions

We welcome:
- Algorithm implementations
- Experimental validation
- Mathematical proofs
- Documentation improvements
- Multi-modal extensions

## Development Guidelines

1. Follow the existing code structure
2. Document all changes with validation status
3. Include limitations in every contribution
4. Maintain the scientific rigor standards

## Repository Structure

The framework repository contains:
- Axioms/ — 16 axiom specifications
- Theory/ — Cognitive Centroid theory
- Mathematics/ — 292 mathematical objects
- Algorithms/ — Algorithm specifications
- Validation/ — Acceptance criteria
- Experiments/ — Formal experiment catalog (38)
- Benchmarks/ — Benchmark suite (24)
